"dns": {
        "servers": [
            {
                "address": "8.8.8.8",
                "port": 53,
                "domains": [
                    "domain:netflix.com",

已上表示已在配置文件中写入内置DNS并指定netflix相关流量走这个DNS的规则，但这时不一定有效，看第二段

第二段

"outbounds": [
        {
            "protocol": "freedom",
            "settings": {
                "domainStrategy": "UseIP"
            },
            "tag": "direct"
        },

已上表示严格执行从上到下进行DNS查询，内置规则优先

注意

1.修改时不要复制粘贴，避免某些小白复制到不可视字符

2.自行举一反三，类似配置都适用

3.修改完成要重启服务端
